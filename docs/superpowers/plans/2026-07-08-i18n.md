# Internacionalização (i18n) Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) ou superpowers:executing-plans para implementar este plano passo a passo. Utilize checkboxes (`- [ ]`) para monitorar o progresso.

**Goal:** Implementar a Prioridade 6 (Internacionalização). Adicionar suporte para múltiplos idiomas (Português, Inglês e Espanhol) em toda a interface do Marvel Champions Helper, permitindo que a comunidade global possa usar a ferramenta.

**Tech Stack:** `i18next`, `react-i18next`, `i18next-browser-languagedetector`.

---

### Task 1: Instalação das Dependências

**Command:**
```bash
npm install i18next react-i18next i18next-browser-languagedetector
```

- [ ] Instalar os pacotes necessários para o i18n.

---

### Task 2: Configuração Inicial do i18n

**Files:**
- Criar: `src/i18n.js`
- Criar diretórios: `src/locales/pt`, `src/locales/en`, `src/locales/es`
- Criar arquivos: `src/locales/{lang}/translation.json`
- Modificar: `src/main.jsx` (Importar o `i18n.js`)

- [ ] **Step 1:** Criar a estrutura de pastas e os 3 arquivos `translation.json` básicos contendo chaves como "nav.home", "nav.history", etc.
- [ ] **Step 2:** Configurar o `src/i18n.js` para usar o detector de idioma e carregar os resources estaticamente (já que o app é simples e sem backend).

---

### Task 3: Traduzir a Interface Principal (Navbar e Rotas Comuns)

**Files:**
- Modificar: `src/App.jsx`

- [ ] Substituir os textos manuais do Menu (Navegação) pelo Hook `useTranslation` (ex: `t('nav.history')`).
- [ ] Adicionar um seletor de idiomas (Ícone de Globo ou dropdown) na Navbar (possivelmente ao lado do botão de login/avatar).

---

### Task 4: Traduzir Conteúdos Dinâmicos e Páginas

**Files:**
- Modificar páginas chave progressivamente (`Dashboard.jsx`, `History.jsx`, `Tracker.jsx`, etc.) usando `const { t } = useTranslation();`.

*(Nota para os workers: Não precisa traduzir 100% de todo o banco de dados das cartas agora, mas garantir que a "casca" do sistema, menus, modais, painéis do tracker e labels de formulário estejam integrados com `t()`).*

---

### Task 5: Commit e Validação

```bash
git checkout -b feature/i18n preview
npm run lint
git add .
git commit -m "feat: implement i18n and multi-language support"
```
