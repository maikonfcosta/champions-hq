# Champions HQ

![Champions HQ Logo](./public/logo.jpg)

**Seu quartel-general definitivo para Marvel Champions LCG.**

## Sobre o Projeto
O **Champions HQ** é uma web application moderna, leve e Mobile-First criada para ajudar os jogadores do Card Game *Marvel Champions* a gerenciar sua coleção, buscar decks e gerar partidas caóticas.

Todo o banco de dados do *MarvelCDB* (de 2019 a 2026) está embutido no projeto, garantindo pesquisas e *match* de cartas em velocidade imediata, 100% offline-ready.

## Funcionalidades
- **Minha Coleção**: Ative e desative os pacotes de heróis e caixas que você possui fisicamente.
- **Banco de Decks**: Explore milhares de decks criados pela comunidade. Inclui o filtro inteligente "Apenas Cartas que Possuo".
- **Gerador de Caos**: Role os dados para um encontro 100% aleatório, selecionando um Vilão, um Herói e um Aspecto com animações responsivas.
- **Guia de Regras**: Dicionário interativo de palavras-chave e fluxo detalhado da Fase do Vilão.

## Tecnologias Usadas
- **React + Vite**
- **Vanilla CSS (Arquitetura Glassmorphism)**
- **Scrapers em Node.js (Raspagem Histórica)**
- **Lucide Icons**

## Como Executar Localmente

```bash
# Instalar dependências
npm install

# Rodar o servidor de desenvolvimento
npm run dev

# Criar a Build de Produção
npm run build
```
